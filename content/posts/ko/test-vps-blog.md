# VPS 자동 설치 테스트

## 설치 완료 환경

이 포스트는 **blog CLI 도구**를 통해 자동으로 발행되었습니다.

### 기술 스택
- **VPS**: Vultr Tokyo Region
- **도메인**: beomanro.com
- **SSL**: Cloudflare Origin Certificate
- **WordPress**: WordOps 자동 설치
- **테마**: Avada Premium Theme
- **캐싱**: Redis
- **서버**: Nginx + PHP 8.1

### 자동화 기능

이 시스템은 다음을 자동화합니다:

1. 마크다운으로 로컬에서 포스트 작성
2. CLI 명령어로 WordPress에 자동 발행
3. 광고 코드 자동 삽입 (AdSense)
4. 이미지 자동 업로드 및 최적화
