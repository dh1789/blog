# About - 소개

안녕하세요! 이 블로그를 방문해 주셔서 감사합니다.

## 블로그 소개

이 블로그는 **개발 자동화와 생산성 향상**에 관심이 많은 개발자가 운영하는 기술 블로그입니다. 특히 WordPress 블로그 자동화, AI 기반 콘텐츠 생성, 그리고 개발 도구 제작 경험을 공유합니다.

### 주요 콘텐츠

- **WordPress 자동화**: 마크다운 기반 블로그 포스팅 자동화, REST API 활용
- **AI 활용**: Claude AI, DALL-E 등을 활용한 콘텐츠 및 이미지 생성
- **개발 도구**: CLI 도구, 생산성 향상을 위한 자동화 스크립트
- **SEO 최적화**: Rank Math, Yoast SEO 등 SEO 플러그인 활용 경험
- **기술 튜토리얼**: 실제 프로젝트 경험을 바탕으로 한 실용적 가이드

## 블로그 기술 스택

이 블로그는 다음 기술로 구축되었습니다:

- **CMS**: WordPress 6.7+ with Avada Theme
- **SEO**: Rank Math SEO Plugin
- **자동화 도구**: [@blog/cli](https://github.com/dh1789/blog) - 자체 개발 CLI 도구
- **AI 통합**: Claude API, OpenAI DALL-E 3
- **호스팅**: (호스팅 정보는 추후 업데이트)

## 작성자 소개

개발자이자 자동화 도구 제작자로, **반복 작업을 자동화하여 더 중요한 일에 집중하는 것**을 추구합니다.

주요 관심 분야:
- 🤖 AI 기반 콘텐츠 생성 및 자동화
- ⚙️ 개발 도구 및 CLI 제작
- 🚀 생산성 향상 및 워크플로우 최적화
- 📝 WordPress 커스터마이징 및 플러그인 개발
- 🔍 SEO 및 블로그 수익화

## 블로그 프로젝트: @blog/cli

이 블로그의 모든 콘텐츠는 자체 개발한 **@blog/cli** 도구를 통해 관리됩니다.

### 주요 기능

- **AI 초안 생성**: Claude API를 활용한 블로그 포스트 초안 자동 생성
- **마크다운 기반 작성**: 로컬에서 편하게 마크다운으로 작성
- **자동 발행**: WordPress REST API를 통한 자동 발행
- **SEO 최적화**: 자동 메타 태그 생성 및 키워드 밀도 체크
- **AdSense 통합**: 광고 코드 자동 삽입
- **실시간 프리뷰**: 작성 중인 포스트를 실시간으로 미리보기
- **이미지 생성**: DALL-E 3를 활용한 AI 이미지 생성
- **트렌드 모니터링**: Reddit, Hacker News 트렌드 수집

자세한 내용은 [GitHub 리포지토리](https://github.com/dh1789/blog)를 참고해 주세요.
