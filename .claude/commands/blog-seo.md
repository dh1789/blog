# /blog-seo - SEO 분석

마크다운 파일의 SEO 점수를 분석하고 개선 제안을 제공합니다.

## 사용자 입력

$ARGUMENTS

## 실행 명령어

```bash
node packages/cli/dist/index.mjs analyze-seo <파일경로> --verbose
```

## 분석 항목

### 점수 카테고리
| 카테고리 | 최대 점수 | 설명 |
|---------|----------|------|
| 제목 길이 | 5 | 60자 이하 권장 |
| 요약 길이 | 5 | 10-300자 |
| 콘텐츠 길이 | 10 | 적정 길이 |
| 키워드 밀도 | 30 | 0.5-2.5% |
| 섹션 분포 | 20 | 50%+ 섹션에 키워드 |
| 제목 키워드 | 15 | 제목에 키워드 포함 |
| 요약 키워드 | 15 | 요약에 키워드 포함 |

### 키워드 밀도 기준
- **목표**: 각 태그 키워드 0.5% - 2.5%
- **최소**: 각 키워드 본문에 5회 이상 출현 권장

## 결과 해석

- **90점 이상**: 우수 - 발행 권장
- **70-89점**: 양호 - 발행 가능
- **70점 미만**: 개선 필요 - 발행 금지

## 개선 제안 예시

```
💡 개선 제안:
  1. 'RAG' 키워드를 "Introduction" 섹션에 3회 추가 권장
  2. 키워드가 없는 섹션이 많습니다. "Setup" 섹션에 키워드를 추가하세요
```
