## 🎯 최종 호스팅 선택

조사 결과를 바탕으로 다음 두 옵션을 추천합니다:

### 옵션 1: SiteGround (추천) ⭐⭐⭐⭐⭐

**장점**:
- 우수한 성능 (592ms, Google Cloud 기반)
- 포괄적인 WordPress 기능
- 안정적인 REST API 지원
- 자동 백업 및 무료 CDN/SSL

**가격**: 첫 해 프로모션, 갱신 후 $17.99/월

**추천 플랜**: StartUp 또는 GrowBig

### 옵션 2: Bluehost (예산 최소화) ⭐⭐⭐⭐

**장점**:
- 매우 저렴한 초기 가격 ($1.99/월)
- WordPress 공식 추천
- 초보자 친화적
- 무료 도메인 1년

**가격**: $1.99/월 (초기), 갱신 후 $2.95-13.95/월

**추천 플랜**: Basic (Starter) 또는 Plus (Business)

## 📝 계정 생성 단계

### SiteGround 계정 생성

1. **웹사이트 접속**
   - https://www.siteground.com/

2. **플랜 선택**
   - "WordPress Hosting" 클릭
   - **StartUp** 또는 **GrowBig** 플랜 선택
   - GrowBig 추천 이유:
     - 무제한 웹사이트
     - 30GB 스토리지
     - On-Demand Backup
     - 캐싱 및 CDN 포함

3. **도메인 설정**
   - 새 도메인 등록 또는 기존 도메인 사용
   - 도메인 옵션:
     - 새 도메인: `yourblog.com` (연간 $15-20)
     - 기존 도메인: 보유 중인 도메인 연결
     - 임시: SiteGround 제공 임시 도메인 사용 (나중에 변경 가능)

4. **계정 정보 입력**
   - 이메일 주소
   - 비밀번호
   - 연락처 정보
   - 결제 정보

5. **추가 옵션 선택**
   - SG Site Scanner: 권장 (보안)
   - SG Site Scanner + Anti-bot AI: 선택사항
   - Domain Privacy: 권장 (개인정보 보호)

6. **결제 완료**
   - 신용카드 또는 PayPal
   - 계약 기간 선택 (1년, 2년, 3년)
   - 장기 계약 시 할인율 증가

7. **계정 생성 완료**
   - 환영 이메일 수신
   - 호스팅 대시보드 접속

## 📋 도메인 옵션 가이드

### 옵션 1: 새 도메인 등록 (권장)

**장점**:
- 브랜드 정체성 확립
- SEO 이점
- 전문적 이미지

**도메인 선택 팁**:
- 짧고 기억하기 쉬운 이름
- .com 도메인 우선 (국제적)
- .kr 또는 .co.kr (한국 타겟 시)
- 키워드 포함 고려 (SEO)

**예시**:
- `techblog-ai.com`
- `dev-automation.com`
- `kodeblog.com`

**비용**: $15-20/년

---

### 옵션 2: 기존 도메인 사용

이미 보유한 도메인이 있다면 호스팅에 연결

**단계**:
1. 호스팅 계정 생성 시 "Use a domain I already own" 선택
2. 도메인 입력
3. 네임서버 변경 (도메인 등록업체에서)
   - 호스팅 제공업체의 네임서버로 변경
   - 전파 시간: 24-48시간

---

### 옵션 3: 임시 도메인 사용 (나중에 변경)

**장점**:
- 즉시 시작 가능
- 비용 절약 (초기)
- 나중에 도메인 추가 가능

**단점**:
- 임시 URL (예: `temp123.siteground.com`)
- 브랜드 정체성 없음
- 나중에 도메인 변경 필요

**권장**: 테스트 목적으로만 사용, 빠른 시일 내 도메인 등록

## 🚨 중요 사항

### 보안

1. **강력한 비밀번호 사용**
   - 12자 이상
   - 대소문자, 숫자, 특수문자 조합
   - 비밀번호 관리자 사용 권장 (1Password, LastPass)

2. **2FA (Two-Factor Authentication) 활성화**
   - 호스팅 계정 보안 강화
   - Google Authenticator 또는 Authy 사용

3. **환영 이메일 보관**
   - 계정 정보, 네임서버, 접속 URL 포함
   - 안전한 곳에 보관

### 비용

1. **초기 프로모션 가격**
   - 첫 계약 기간만 적용
   - 갱신 시 정가로 청구

2. **자동 갱신 설정**
   - 기본적으로 활성화
   - 갱신 전 이메일 알림 확인
   - 필요 시 플랜 변경 또는 취소

3. **환불 정책 확인**
   - SiteGround: 30일 환불 보장
   - Bluehost: 30일 환불 보장
   - 도메인 등록비는 환불 불가
