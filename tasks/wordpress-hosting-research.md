# WordPress 호스팅 옵션 조사 (2025년)

Epic 6.0 - Task 6.1의 첫 번째 서브태스크: WordPress 호스팅 서비스 비교 분석

**조사 날짜**: 2025-10-28
**목적**: @blog/cli 프로젝트의 프로덕션 배포를 위한 최적의 WordPress 호스팅 선택

## 📋 조사 대상 호스팅 서비스

1. **Bluehost** - 초보자 친화적, WordPress 공식 추천
2. **SiteGround** - 성능 우수, Google Cloud 기반
3. **WP Engine** - 프리미엄 관리형 호스팅
4. **DigitalOcean** - 개발자 친화적, VPS 기반
5. **ScalaHosting** - 한국(서울) 서버 제공

## 1️⃣ Bluehost

### 가격 (2025년)

**공유 호스팅**:
- **Starter (Basic)**: $1.99/월 (초기), $2.95-13.95/월 (갱신)
- **Business (Plus)**: $3.95/월 (초기), $23.88/년 (1년 계약)
- **Choice Plus**: 중간 플랜
- **Pro**: $13.95/월

**관리형 WordPress 호스팅**:
- $19.95-49.95/월

**WooCommerce 호스팅**:
- $7.45-14.99/월

**Cloud 호스팅**:
- $75/월 (Cloud 1: $30/월)

### 주요 특징

✅ **포함 사항**:
- 무료 도메인, SSL, 마이그레이션
- 1-클릭 WordPress 설치 및 자동 업데이트
- 99.9% 업타임 보장
- NVMe 스토리지 (빠른 속도)
- 일일 자동 백업 (오프사이트)
- WordPress 전문가 24/7 지원

### 성능

- **로드 타임**: 0.44s (2025년 7월 미국 테스트)
- **장점**: 대부분의 호스팅보다 빠른 로드 타임

### 장단점

**장점**:
- ✅ WordPress 공식 추천
- ✅ 초보자 친화적 인터페이스
- ✅ 매우 저렴한 초기 가격
- ✅ 무료 도메인 1년 제공
- ✅ WordPress Academy 교육 자료

**단점**:
- ❌ 갱신 시 가격 급등 (프로모션 가격은 첫 계약만)
- ❌ 장기 비용이 높을 수 있음

### 추천 대상

- WordPress 초보자
- 저렴한 시작 비용을 원하는 사용자
- 간편한 설정을 선호하는 사용자

## 2️⃣ SiteGround

### 가격 (2025년)

- **첫 해**: 프로모션 가격
- **갱신 후**: $17.99/월 (최저 공유 호스팅)

### 주요 특징

✅ **포함 사항**:
- Google Cloud 인프라 기반
- 자동 업그레이드 및 일일 백업
- 내장 WordPress 캐싱
- 무료 CDN 및 SSL
- 1-클릭 스테이징 환경
- GIT 버전 관리 지원

### 성능

- **완전 로드 시간**: 592ms
- **Largest Contentful Paint (LCP)**: 498ms
- **First Contentful Paint (FCP)**: 336ms
- **스트레스 테스트**: 11,600 요청에서 HTTP 실패 없음
- **평가**: 상위 티어 공유 호스팅 성능

### 장단점

**장점**:
- ✅ 뛰어난 성능 (WP Engine과 비슷한 수준)
- ✅ Google Cloud 기반
- ✅ 포괄적인 WordPress 기능
- ✅ 우수한 업타임 및 속도

**단점**:
- ❌ 첫 해 이후 가격 급등 ($17.99/월)
- ❌ 장기적으로 비용 부담

### 추천 대상

- 성능을 중시하는 사용자
- 중급 WordPress 사용자
- Google Cloud 인프라를 선호하는 사용자

## 3️⃣ WP Engine

### 가격 (2025년)

- **Startup**: $20-30/월 (1 사이트, 25,000 방문, 10GB 스토리지, 50GB 대역폭)
  - 이전: $35/월 → 현재: $30/월
  - 연간: $350/년 → $300/년
- **Growth**: $115/월 (10 사이트, 이전 5 사이트에서 증가)
- **Scale**: $290/월 (30 사이트, 이전 15 사이트에서 증가)
- **프리미엄 커스텀 플랜**: $58/월 이상

### 주요 특징

✅ **포함 사항**:
- 자동 보안 업데이트
- 일일 백업 및 1-클릭 복원
- 자동 캐싱
- 1-클릭 스테이징 환경
- 60일 환불 보장 (업계 표준의 2배)
- WordPress 전문 관리형 서비스

### 성능

- **TTFB (Time To First Byte)**: 354ms (17개 제공업체 중 최고)
- **업타임**: 99.99%
- **로드 테스트 응답 시간**: 19ms
- **평가**: 엘리트 급 성능

### 장단점

**장점**:
- ✅ 최고 수준의 성능 (354ms TTFB)
- ✅ 99.99% 안정성
- ✅ 프리미엄 관리형 서비스
- ✅ 60일 환불 보장
- ✅ 자동 보안 및 백업

**단점**:
- ❌ 높은 가격 (최소 $20-30/월)
- ❌ 소규모 블로그에는 과잉 사양
- ❌ 초보자에게는 비용 부담

### 추천 대상

- 중대형 WordPress 사이트
- 성능과 안정성을 최우선하는 사용자
- 관리 부담을 최소화하고 싶은 사용자
- 전문적인 비즈니스 사이트

---

## 4️⃣ DigitalOcean

### 가격 (2025년)

- **기본 Droplet**: $4/월부터
- **확장 가능**: 언제든지 상위 플랜으로 업그레이드 가능

### 주요 특징

✅ **설치 방법**:
1. **1-클릭 WordPress 앱**: Apache, MySQL, PHP (LAMP 스택) 포함
2. **수동 LEMP 설정**: 완전한 커스터마이징
3. **Cloudways 관리형**: 자동 업데이트, 보안, 확장성

✅ **장점**:
- VPS 기반 완전한 서버 제어권
- 저렴한 시작 비용
- 유연한 확장성
- 개발자 친화적

### 성능

- 서버 리소스에 따라 다름 (사용자 설정)
- 최적화 가능성 높음 (수동 설정)

### 장단점

**장점**:
- ✅ 매우 저렴한 시작 가격 ($4/월)
- ✅ 완전한 서버 제어권
- ✅ 유연한 확장성
- ✅ 개발자에게 이상적

**단점**:
- ❌ 기술적 지식 필요 (서버 관리, 보안, 성능 최적화)
- ❌ 관리형이 아니면 모든 것을 직접 처리
- ❌ 초보자에게는 어려움
- ❌ 시간 투자 필요

### 추천 대상

- 서버 관리 경험이 있는 개발자
- 완전한 제어권을 원하는 사용자
- 비용 최소화를 원하는 기술적 사용자
- 커스터마이징이 중요한 프로젝트

## 5️⃣ ScalaHosting (한국 서버)

### 가격

- 정보 제한적 (웹 검색 결과에서 상세 가격 미확인)

### 주요 특징

✅ **한국 사용자 최적화**:
- **서울 서버 위치**: 20ms 미만 지연시간
- **로컬 IP**: 한국 지역 SEO 이점
- **빠른 로드 속도**: 한국 사용자를 위한 최적화

### 장단점

**장점**:
- ✅ 서울 서버로 한국 사용자에게 최적 성능
- ✅ 로컬 SEO 이점
- ✅ 매우 낮은 지연시간 (20ms 미만)

**단점**:
- ❌ 가격 정보 부족
- ❌ 국제적 인지도 낮음
- ❌ 영어 리소스 제한적

### 추천 대상

- 한국 타겟 오디언스를 가진 사이트
- SEO에서 로컬 IP가 중요한 사이트
- 한국어 지원이 필요한 사용자

## 🏆 최종 추천

### 우리 프로젝트에 가장 적합한 호스팅: **SiteGround**

**선택 근거**:
1. **성능**: 592ms 로드 타임, Google Cloud 기반
2. **기능**: 자동 백업, 캐싱, 스테이징, GIT 지원
3. **안정성**: 상위 티어 업타임 및 속도
4. **가격**: 첫 해 프로모션 가격, 갱신 후 $17.99/월 (합리적)
5. **확장성**: 블로그 성장에 따라 플랜 업그레이드 가능
6. **REST API**: WordPress REST API 안정적 지원

### 대안 (예산 최소화 시): **Bluehost**

초기 비용을 최소화하고 싶다면 Bluehost ($1.99/월)를 선택하고, 블로그 성장 후 SiteGround 또는 WP Engine으로 마이그레이션 고려
