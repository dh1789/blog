# 프롬프트 템플릿

이 디렉토리는 AI 블로그 포스트 생성을 위한 프롬프트 템플릿을 포함합니다.

## 사용 가능한 템플릿

### 1. blog-post.txt
일반 블로그 포스트 작성용 템플릿입니다.

**용도**: 기술 트렌드, AI 도구 소개, 개발 팁 등
**스타일**: 친근하고 실용적
**구조**: 도입부(10%) → 본문(75%) → 결론(15%)

**사용 예시**:
```bash
blog draft create "ChatGPT 활용법" "AI,ChatGPT,생산성" -t blog-post
```

### 2. tutorial.txt
단계별 튜토리얼 작성용 템플릿입니다.

**용도**: 기술 학습, 실습 가이드, 프로젝트 튜토리얼
**스타일**: 교육적이고 단계적
**구조**: 개요(10%) → 준비(10%) → 구현(60%) → 테스트(10%) → 마무리(10%)

**사용 예시**:
```bash
blog draft create "Next.js 시작하기" "Next.js,React,튜토리얼" -t tutorial
```

### 3. review.txt
제품/서비스 리뷰용 템플릿입니다.

**용도**: 도구 리뷰, 서비스 평가, 비교 분석
**스타일**: 객관적이고 균형잡힌 분석적
**구조**: 서론(10%) → 개요(15%) → 평가(50%) → 장단점(15%) → 결론(10%)

**사용 예시**:
```bash
blog draft create "GitHub Copilot 리뷰" "Copilot,AI코딩,개발도구" -t review
```

## 템플릿 변수

모든 템플릿은 다음 변수를 지원합니다:

- `{TOPIC}` - 주제
- `{KEYWORDS}` - 키워드 (쉼표로 구분)
- `{WORDS}` - 타겟 단어 수
- `{LANGUAGE}` - 언어 (ko 또는 en)

## 커스텀 템플릿 추가

1. 이 디렉토리에 `.txt` 파일 생성
2. 위 변수 형식 사용 (`{VARIABLE}`)
3. 다음 명령어로 사용:
   ```bash
   blog draft create "주제" "키워드" -t 템플릿명
   ```

## 템플릿 작성 가이드

### 기본 구조
```
당신은 [역할] 전문가입니다.

## 요구사항
**주제**: {TOPIC}
**키워드**: {KEYWORDS}
**타겟 단어 수**: {WORDS} 단어
**언어**: {LANGUAGE}

## 타겟 독자
[독자층 설명]

## 작성 스타일
- **톤**: [톤 설명]
- **접근법**: [접근 방법]
- **구조**: [구조 설명]

[상세 구조]

## 작성 가이드라인

```
